# User Documentation

This documentation is intended for end users and administrators of the FormFiller system.

## Contents

- [Getting Started](./getting-started.md) - First steps in using the system
- [Form Configuration](./form-configuration.md) - Creating and customizing forms
- [Admin Guide](./admin-guide.md) - System administration

## System Overview

FormFiller is a dynamic low-code framework that currently provides:

- **Creating forms** - Based on JSON configuration
- **Collecting data** - Storing and managing submitted data
- **Managing data grids** - Tabular data display
- **Managing tree structures** - Hierarchical data
- **Access control** - Who can see and edit what

## Quick Overview

### Login

1. Open the application URL
2. Click the "Login" button
3. Use your Google account or local credentials

### Navigation

- **Home**: Available forms and features
- **Forms**: Create new records
- **Results**: View and edit existing data
- **Profile**: User settings

### Filling Out Forms

1. Select the form from the list
2. Fill in the fields
3. Click the "Save" button

### Viewing Data

1. Open the "Results" view
2. Use filters and search
3. Click a row to view details

## FAQ

### How can I modify an already submitted record?

1. Open the "Results" view
2. Find the record
3. Click the "Edit" button
4. Modify the data
5. Save

### How can I export data?

1. Open the "Results" view
2. Apply filters (optional)
3. Click the "Export" button
4. Choose the format (Excel/PDF)

### What should I do if I get an error?

1. Check that all required fields are filled
2. Check the error messages below the fields
3. If the problem persists, contact the administrator

## Contact

For technical issues, contact the system administrator.
